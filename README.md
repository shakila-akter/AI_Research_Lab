## 📌 Project Overview

This project is a **modern AI Research Lab Website** built using **HTML, CSS, and Bootstrap**.
It is designed as a **single-page responsive website** that showcases the structure of a real university AI lab.

The website presents key information such as research areas, ongoing projects, publications, team members, and contact details in a clean and professional layout.

---

## ✨ Features

* 🏠 **Home Section** – Introduction to the AI Lab
* ℹ️ **About Section** – Vision, mission, and goals
* 🔬 **Research Section** – 6 research domains using card layout
* 💡 **Projects Section** – Active and completed projects with tags
* 📚 **Publications Section** – Papers grouped by year
* 👨‍🏫 **Team Section** – Faculty and student members (responsive grid)
* 📞 **Contact Section** – Contact form and map placeholder
